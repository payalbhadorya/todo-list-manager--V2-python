# To-Do List Manager (Python)

A menu-driven To-Do List Manager built using Python.

## Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using JSON file handling

## Technologies Used
- Python
- JSON

## How to Run
1. Clone or download the repository
2. Run `to_do_manager_v2.py`
3. Follow the menu options
